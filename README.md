# Тестовое задание

Задача: Рефакторинг компонента UserTable
Необходимо произвести рефакторинг монолитного компонента таблицы пользователей UserTable, написанного на Options API.
Требования
1. Переписать на Composition API
2. Добавить TypeScript — типизировать все компоненты, props, события
3. Разбить на подкомпоненты
4. Вынести логику в composables


### Комментарии
Используется стандартный проект Vite + Vue + TypeScript.

Есть возможности для улучшения (но это тестовое задание).

Возможно есть ошибки.

## Настройка проекта
```
npm install
```

### Запуск для разработки
```
npm run dev
```

### Сборка на прод
```
npm run build
```
